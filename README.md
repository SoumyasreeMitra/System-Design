# Document Editor (Java)

A simple document editor project implemented using Java.

## Features
- Add text
- Insert images
- New line and tab support
- Render document
- Save document to file

## Concepts Used
- Object-Oriented Programming
- Interfaces & Polymorphism
- SOLID Principles
- File Handling

## How to Run
1. Compile all `.java` files
2. Run `DocumentEditorClient`
